# 1. اپنی ریپوزٹری کی تازہ ترین حالت چیک کریں
git pull origin main

# 2. اگر آپ نے فائل میں تبدیلیاں کر لی ہیں، تو اسے 'اسٹیج' کریں
git add README.md

# 3. تبدیلی کو محفوظ (Commit) کریں
git commit -m "README.md فائل کو اپڈیٹ کر دیا ہے - نور پراجیکٹ"

# 4. کوڈ واپس GitHub پر اپلوڈ کریں
git push origin main
